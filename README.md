## 🇺🇸 English Documentation

**A powerful, browser-based IDE with zero backend requirements.**

### Features
- IDE Interface: Auto-problem panel and error linting.
- Zero Backend: Runs entirely in the browser using IndexedDB.
- Multi-language Support: Dynamic detection and switching of 30 languages.
- Python & JS Execution: Instantly run code in the browser via WebAssembly.

### Keyboard Shortcuts
- `Ctrl + S`: Save / Export
- `Ctrl + Enter`: Run Code
- `Ctrl + Space`: Autocomplete
- `Ctrl + F`: Search

### Project Structure
```
WebPad_Project/
├── index.html        (Main Shell)
├── css/style.css     (Styles)
├── js/
│   ├── editor.js     (CodeMirror & Tab Logic)
│   ├── filesystem.js (IndexedDB Virtual Storage)
│   └── i18n.js       (Multi-language Engine)
└── locales/          (30+ JSON Translation Files)
```
