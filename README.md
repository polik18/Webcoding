## 繁體中文 (Traditional Chinese)

**WebPad++** 是一個輕量級、功能強大且完全運行於前端瀏覽器的現代化線上 IDE 與視覺化編輯器。整個專案僅由單一 HTML 檔案構成，無需任何伺服器或後端環境，雙擊即可在任何現代瀏覽器中流暢運行。

### ✨ 核心特色功能

1. **🌍 智慧多國語言 (i18n)**
   - 內建英文、繁體中文、西班牙文、法文、日文。自動偵測瀏覽器語系，並具備記憶功能的下拉選單。
2. **🌗 雙模式無縫切換**
   - **程式碼模式：** 支援語法高亮與行號。
   - **視覺化模式：** 類似 Word 的編輯器，支援文字樣式與完整的**表格增刪操作**。切換回程式碼時會自動排版。
3. **🐛 專業級即時除錯 (Linting)**
   - 內建底部開發者面板，整合 HTMLHint, CSSLint, JSHint，點擊錯誤訊息自動跳轉行數。
4. **▶️ 多語言跨界執行 (Run)**
   - 支援 HTML 新分頁預覽、JavaScript 終端機回傳，以及基於 **WebAssembly (Pyodide)** 的純瀏覽器 Python 執行引擎。
5. **📁 深度本地檔案整合**
   - 支援拖曳開檔、點擊修改檔名/副檔名，以及防呆降級存檔機制。
6. **📱 全面響應式設計 (RWD)**
   - 電腦與手機皆可完美適應。

### 🚀 快速開始

無需安裝！下載 `index.html`，用任何現代瀏覽器開啟即可開始使用。

## English

**WebPad++** is a lightweight, powerful, and fully browser-based modern online IDE and visual editor. Built entirely within a single HTML file, it requires no server or backend and runs smoothly in any modern browser.

### ✨ Core Features

1. **🌍 Smart Multilingual (i18n)**
   - Built-in support for EN, ZH, ES, FR, JA. Auto-detects browser language and features a memory-enabled dropdown selector.
2. **🌗 Seamless Dual Modes**
   - **Code Mode:** Features syntax highlighting and line numbers.
   - **Visual Mode:** A Word-like WYSIWYG editor supporting rich text and **full table operations (add/remove rows & cols)**. Auto-formats HTML upon switching back.
3. **🐛 Pro-Level Linting**
   - Built-in developer panel integrating HTMLHint, CSSLint, and JSHint. Click on an error to jump directly to the line.
4. **▶️ Cross-Language Execution (Run)**
   - Supports HTML live preview in a new tab, JavaScript console output, and a true Python execution engine via **WebAssembly (Pyodide)**.
5. **📁 Deep Local File Integration**
   - Supports drag-and-drop, click-to-rename for changing extensions, and a fail-safe fallback saving mechanism.
6. **📱 Fully Responsive Design (RWD)**
   - Adapts perfectly to both desktop and mobile screens.

### 🚀 Quick Start

Zero setup! Download `index.html` and open it in any modern browser to begin.
