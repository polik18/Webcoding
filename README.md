## 🌟 Overview

WebPad++ is a next-generation Integrated Development Environment (IDE) that runs **100% in your web browser**. Without needing a backend, databases, or local installations, you can write code, run Python, build websites, manage spreadsheets, and extract text from PDFs—all locally on your machine with zero latency.

## ✨ Key Features

### 💻 Code Editing & Execution
- **Intelligent Editor**: Powered by Ace Editor with syntax highlighting, IntelliSense, and bracket matching.
- **Instant Execution**: 
  - Run **HTML/CSS/JS** with a live side-by-side Preview.
  - Run **Python** locally directly in the browser via Pyodide.
- **Built-in Linter & Formatter**: Real-time syntax error detection and one-click beautify for HTML, CSS, and JS.
- **Code Diff**: Compare any two files side-by-side with our visual merge viewer.

### 📄 Professional Document Suite
- **Markdown & Visual Sync**: Seamlessly toggle between Markdown code and a Rich Text (WYSIWYG) visual editor.
- **Spreadsheet Engine**: Open, edit, sort, and apply mathematical formulas (e.g., `=SUM(A1:A5)`) to `.xlsx` and `.csv` files.
- **PDF & Word Tools**: View PDFs, edit DOCX files in rich-text mode, and extract text via built-in parsers.
- **Exporting**: Convert and download files across DOCX, XLSX, CSV, and PDF formats.

### 🌐 Advanced Utilities
- **Folder Drag & Drop**: Drag entire project directories from your OS straight into WebPad++.
- **SEO Tools**: Built-in visual generators for `sitemap.xml` and `robots.txt` to boost your site's search ranking.
- **Image OCR**: Upload or snap a photo of a document and automatically extract the text using Tesseract.js.
- **QR Code Suite**: Generate QR codes instantly or decode them using your webcam.

## 🚀 How to Use

1. **Getting Started**: Simply open `index.html` in any modern web browser.
2. **File Management**: Click the `☰` menu to open the File Explorer. Right-click to create files, or simply drag and drop folders into the window.
3. **Running Code**: Open any script and click the ▶️ **Run** button or press `Ctrl+Enter`.
4. **Tools & SEO**: Use the top toolbar to access the OCR tool, QR Code scanner, or Sitemap/Robots generator.
5. **Languages**: WebPad++ supports 30 global languages! Use the dropdown menu in the top right to switch instantly.

## 🔐 Privacy & Security

WebPad++ operates entirely client-side. Your code, documents, and data never leave your computer. Everything is stored persistently using your browser's native IndexedDB. You can click **Reset All** to wipe all data securely.
