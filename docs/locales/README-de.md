## 🌟 Übersicht

WebPad++ ist eine integrierte Entwicklungsumgebung (IDE) der nächsten Generation, die **100 % in Ihrem Webbrowser** läuft. Ohne Backend, Datenbanken oder lokale Installationen können Sie Code schreiben, Python ausführen, Websites erstellen, Tabellenkalkulationen verwalten und Text aus PDFs extrahieren – alles lokal auf Ihrem Computer ohne Latenz.

## ✨ Hauptmerkmale

### 💻 Codebearbeitung und -ausführung
- **Intelligenter Editor**: Unterstützt von Ace Editor mit Syntaxhervorhebung, IntelliSense und Klammerabgleich.
- **Sofortige Ausführung**: 
  - Führen Sie **HTML/CSS/JS** mit einer Live-Vorschau nebeneinander aus.
  - Führen Sie **Python** lokal direkt im Browser über Pyodide aus.
- **Integrierter Linter & Formatter**: Syntaxfehlererkennung in Echtzeit und Verschönerung mit einem Klick für HTML, CSS und JS.
- **Code-Diff**: Vergleichen Sie zwei beliebige Dateien nebeneinander mit unserem visuellen Zusammenführungs-Viewer.

### 📄 Professionelle Dokumentensuite
- **Markdown und visuelle Synchronisierung**: Wechseln Sie nahtlos zwischen Markdown-Code und einem visuellen Rich-Text-Editor (WYSIWYG).
- **Spreadsheet Engine**: Öffnen, bearbeiten, sortieren und wenden Sie mathematische Formeln (z. B. „=SUM(A1:A5)“) auf „.xlsx“- und „.csv“-Dateien an.
- **PDF- und Word-Tools**: PDFs anzeigen, DOCX-Dateien im Rich-Text-Modus bearbeiten und Text über integrierte Parser extrahieren.
- **Exportieren**: Konvertieren und Herunterladen von Dateien in den Formaten DOCX, XLSX, CSV und PDF.

### 🌐 Erweiterte Dienstprogramme
- **Ordner-Drag & Drop**: Ziehen Sie ganze Projektverzeichnisse von Ihrem Betriebssystem direkt in WebPad++.
- **SEO-Tools**: Integrierte visuelle Generatoren für „sitemap.xml“ und „robots.txt“, um das Suchranking Ihrer Website zu verbessern.
- **Bild-OCR**: Laden Sie ein Foto eines Dokuments hoch oder machen Sie ein Foto davon und extrahieren Sie den Text automatisch mit Tesseract.js.
- **QR Code Suite**: QR-Codes sofort generieren oder mit Ihrer Webcam dekodieren.

## 🚀 Verwendung

1. **Erste Schritte**: Öffnen Sie einfach „index.html“ in einem modernen Webbrowser.
2. **Dateiverwaltung**: Klicken Sie auf das Menü „☰“, um den Datei-Explorer zu öffnen. Klicken Sie mit der rechten Maustaste, um Dateien zu erstellen, oder ziehen Sie Ordner einfach per Drag & Drop in das Fenster.
3. **Code ausführen**: Öffnen Sie ein beliebiges Skript und klicken Sie auf die Schaltfläche ▶️ **Ausführen** oder drücken Sie „Strg+Eingabe“.
4. **Tools und SEO**: Verwenden Sie die obere Symbolleiste, um auf das OCR-Tool, den QR-Code-Scanner oder den Sitemap-/Robots-Generator zuzugreifen.
5. **Sprachen**: WebPad++ unterstützt 30 globale Sprachen! Verwenden Sie das Dropdown-Menü oben rechts, um sofort zu wechseln.

## 🔐 Datenschutz und Sicherheit

WebPad++ arbeitet vollständig clientseitig. Ihr Code, Ihre Dokumente und Daten verlassen niemals Ihren Computer. Alles wird dauerhaft über die native IndexedDB Ihres Browsers gespeichert. Sie können auf **Alle zurücksetzen** klicken, um alle Daten sicher zu löschen.
