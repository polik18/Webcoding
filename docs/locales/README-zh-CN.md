## 🌟 概述

WebPad++ 是下一代集成开发环境 (IDE)，**100% 在 Web 浏览器中运行**。无需后端、数据库或本地安装，您就可以编写代码、运行 Python、构建网站、管理电子表格以及从 PDF 中提取文本 - 所有这些都在您的计算机上以零延迟本地完成。

## ✨ 主要特点

### 💻 代码编辑和执行
- **智能编辑器**：由 Ace Editor 提供支持，具有语法突出显示、IntelliSense 和括号匹配功能。
- **即时执行**： 
  - 通过实时并排预览运行 **HTML/CSS/JS**。
  - 通过 Pyodide 直接在浏览器中本地运行 **Python**。
- **内置Linter & Formatter**：实时语法错误检测和一键美化HTML、CSS和JS。
- **代码差异**：使用我们的可视化合并查看器并排比较任意两个文件。

### 📄 专业文档套件
- **Markdown 和视觉同步**：在 Markdown 代码和富文本 (WYSIWYG) 视觉编辑器之间无缝切换。
- **电子表格引擎**：打开、编辑、排序并将数学公式（例如“=SUM(A1:A5)”）应用到“.xlsx”和“.csv”文件。
- **PDF 和 Word 工具**：查看 PDF、以富文本模式编辑 DOCX 文件，以及通过内置解析器提取文本。
- **导出**：转换和下载 DOCX、XLSX、CSV 和 PDF 格式的文件。

### 🌐 高级实用程序
- **文件夹拖放**：将整个项目目录从操作系统直接拖到 WebPad++ 中。
- **SEO 工具**：内置“sitemap.xml”和“robots.txt”视觉生成器，可提高网站的搜索排名。
- **图像 OCR**：上传或拍摄文档照片并使用 Tesseract.js 自动提取文本。
- **QR 代码套件**：立即生成 QR 代码或使用网络摄像头对其进行解码。

## 🚀 如何使用

1. **入门**：只需在任何现代网络浏览器中打开“index.html”即可。
2. **文件管理**：单击`☰`菜单打开文件资源管理器。右键单击以创建文件，或只需将文件夹拖放到窗口中。
3. **运行代码**：打开任意脚本，然后单击 ▶️ **运行** 按钮或按 `Ctrl+Enter`。
4. **工具和 SEO**：使用顶部工具栏访问 OCR 工具、QR 码扫描仪或站点地图/机器人生成器。
5. **语言**：WebPad++支持30种全球语言！使用右上角的下拉菜单进行即时切换。

## 🔐 隐私与安全

WebPad++ 完全在客户端运行。您的代码、文档和数据永远不会离开您的计算机。所有内容都使用浏览器的本机 IndexedDB 持久存储。您可以单击“**全部重置**”以安全擦除所有数据。
