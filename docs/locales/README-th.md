## 🌟 ภาพรวม

WebPad++ คือ Integrated Development Environment (IDE) ยุคถัดไปที่ทำงาน **100% ในเว็บเบราว์เซอร์ของคุณ** โดยไม่จำเป็นต้องแบ็คเอนด์ ฐานข้อมูล หรือการติดตั้งในเครื่อง คุณสามารถเขียนโค้ด เรียกใช้ Python สร้างเว็บไซต์ จัดการสเปรดชีต และแยกข้อความจาก PDF ทั้งหมดนี้อยู่ในเครื่องของคุณโดยมีเวลาหน่วงเป็นศูนย์

## ✨ คุณสมบัติที่สำคัญ

### 💻 การแก้ไขและการดำเนินการโค้ด
- **ตัวแก้ไขอัจฉริยะ**: ขับเคลื่อนโดย Ace Editor พร้อมการเน้นไวยากรณ์, IntelliSense และการจับคู่วงเล็บเหลี่ยม
- **การดำเนินการทันที**: 
  - เรียกใช้ **HTML/CSS/JS** พร้อมการแสดงตัวอย่างแบบสดเคียงข้างกัน
  - เรียกใช้ **Python** ภายในเครื่องโดยตรงในเบราว์เซอร์ผ่าน Pyodide
- **Linter & Formatter ในตัว**: การตรวจจับข้อผิดพลาดทางไวยากรณ์แบบเรียลไทม์และการตกแต่ง HTML, CSS และ JS ได้ในคลิกเดียว
- **Code Diff**: เปรียบเทียบสองไฟล์แบบเคียงข้างกันกับโปรแกรมดูการรวมภาพของเรา

### 📄 ชุดเอกสารระดับมืออาชีพ
- **Markdown & Visual Sync**: สลับระหว่างโค้ด Markdown และโปรแกรมแก้ไขภาพ Rich Text (WYSIWYG) ได้อย่างราบรื่น
- **เครื่องมือสเปรดชีต**: เปิด แก้ไข จัดเรียง และใช้สูตรทางคณิตศาสตร์ (เช่น `=SUM(A1:A5)`) กับไฟล์ `.xlsx` และ `.csv`
- **เครื่องมือ PDF และ Word**: ดู PDF แก้ไขไฟล์ DOCX ในโหมด Rich-Text และแยกข้อความผ่าน Parsers ในตัว
- **การส่งออก**: แปลงและดาวน์โหลดไฟล์ในรูปแบบ DOCX, XLSX, CSV และ PDF

### 🌐 ยูทิลิตี้ขั้นสูง
- **ลากและวางโฟลเดอร์**: ลากไดเร็กทอรีโปรเจ็กต์ทั้งหมดจากระบบปฏิบัติการของคุณไปยัง WebPad++ โดยตรง
- **เครื่องมือ SEO**: เครื่องมือสร้างภาพในตัวสำหรับ `sitemap.xml` และ `robots.txt` เพื่อเพิ่มอันดับการค้นหาของเว็บไซต์ของคุณ
- **Image OCR**: อัปโหลดหรือถ่ายภาพเอกสารและแยกข้อความโดยอัตโนมัติโดยใช้ Tesseract.js
- **ชุดรหัส QR**: สร้างรหัส QR ทันทีหรือถอดรหัสโดยใช้เว็บแคมของคุณ

## 🛠 รูปแบบที่รองรับ

| รูปแบบ | นำเข้าและดู | แก้ไข | ส่งออก |
|--------|---------|-|--------|
| **HTML/JS/CSS** | ✅ ใช่ | ✅ รหัส & ดูตัวอย่างสด | ✅ ZIP / ไฟล์ |
| **มาร์คดาวน์** | ✅ ใช่ | ✅ รหัส & การซิงค์แบบ WYSIWYG | ✅ นพ. |
| **XLSX / CSV** | ✅ ตารางสเปรดชีต | ✅ สูตรและการเรียงลำดับ | ✅ XLSX / CSV |
| **PDF** | ✅ ผู้ดู | ✅ แยกข้อความ | ✅ PDF / DOCX |
| **DOCX** | ✅ เรนเดอร์ HTML ที่หลากหลาย| ✅ WYSIWYG | ✅ DOCX / PDF |

## 🚀 วิธีใช้งาน

1. **เริ่มต้นใช้งาน**: เพียงเปิด `index.html` ในเว็บเบราว์เซอร์สมัยใหม่
2. **การจัดการไฟล์**: คลิกเมนู `☰` เพื่อเปิด File Explorer คลิกขวาเพื่อสร้างไฟล์ หรือเพียงแค่ลากและวางโฟลเดอร์ลงในหน้าต่าง
3. **การรันโค้ด**: เปิดสคริปต์ใดก็ได้แล้วคลิกปุ่ม ▶️ **Run** หรือกด `Ctrl+Enter`
4. **เครื่องมือและ SEO**: ใช้แถบเครื่องมือด้านบนเพื่อเข้าถึงเครื่องมือ OCR เครื่องสแกนโค้ด QR หรือเครื่องมือสร้างแผนผังไซต์/โรบอต
5. **ภาษา**: WebPad++ รองรับ 30 ภาษาทั่วโลก! ใช้เมนูแบบเลื่อนลงที่ด้านบนขวาเพื่อสลับทันที

## 🔐 ความเป็นส่วนตัวและความปลอดภัย

WebPad++ ทำงานฝั่งไคลเอ็นต์ทั้งหมด รหัส เอกสาร และข้อมูลของคุณจะไม่มีวันออกจากคอมพิวเตอร์ของคุณ ทุกอย่างจะถูกเก็บไว้อย่างต่อเนื่องโดยใช้ IndexedDB ดั้งเดิมของเบราว์เซอร์ของคุณ คุณสามารถคลิก **รีเซ็ตทั้งหมด** เพื่อล้างข้อมูลทั้งหมดอย่างปลอดภัย
