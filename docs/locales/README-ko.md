## 🌟 개요

WebPad++는 **100% 웹 브라우저**에서 실행되는 차세대 통합 개발 환경(IDE)입니다. 백엔드, 데이터베이스 또는 로컬 설치가 필요 없이 코드 작성, Python 실행, 웹 사이트 구축, 스프레드시트 관리, PDF에서 텍스트 추출 등의 모든 작업을 대기 시간 없이 로컬에서 컴퓨터에서 수행할 수 있습니다.

## ✨ 주요 기능

### 💻 코드 편집 및 실행
- **지능형 편집기**: 구문 강조, IntelliSense 및 대괄호 일치 기능을 갖춘 Ace Editor로 구동됩니다.
- **즉시 실행**: 
  - 실시간 병렬 미리보기로 **HTML/CSS/JS**를 실행하세요.
  - Pyodide를 통해 브라우저에서 로컬로 직접 **Python**을 실행합니다.
- **내장 린터 및 포맷터**: HTML, CSS, JS에 대한 실시간 구문 오류 감지 및 원클릭 미화 기능을 제공합니다.
- **코드 차이**: 시각적 병합 뷰어를 사용하여 두 파일을 나란히 비교합니다.

### 📄 전문 문서 모음
- **마크다운 및 시각적 동기화**: 마크다운 코드와 서식 있는 텍스트(WYSIWYG) 시각적 편집기 사이를 원활하게 전환합니다.
- **스프레드시트 엔진**: `.xlsx` 및 `.csv` 파일에 수학 공식(예: `=SUM(A1:A5)`)을 열고, 편집하고, 정렬하고, 적용합니다.
- **PDF 및 Word 도구**: PDF를 보고, 서식 있는 텍스트 모드에서 DOCX 파일을 편집하고, 내장된 파서를 통해 텍스트를 추출합니다.
- **내보내기**: DOCX, XLSX, CSV 및 PDF 형식으로 파일을 변환하고 다운로드합니다.

### 🌐 고급 유틸리티
- **폴더 드래그 앤 드롭**: 전체 프로젝트 디렉토리를 OS에서 WebPad++로 바로 드래그하세요.
- **SEO 도구**: 사이트의 검색 순위를 높이기 위한 `sitemap.xml` 및 `robots.txt`용 내장 시각적 생성기.
- **이미지 OCR**: 문서 사진을 업로드하거나 촬영하고 Tesseract.js를 사용하여 자동으로 텍스트를 추출합니다.
- **QR 코드 제품군**: QR 코드를 즉시 생성하거나 웹캠을 사용하여 디코딩합니다.

## 🚀 사용 방법

1. **시작하기**: 최신 웹 브라우저에서 `index.html`을 열면 됩니다.
2. **파일 관리**: `☰` 메뉴를 클릭하여 파일 탐색기를 엽니다. 마우스 오른쪽 버튼을 클릭하여 파일을 생성하거나 폴더를 창으로 끌어서 놓기만 하면 됩니다.
3. **코드 실행**: 스크립트를 열고 ▶️ **실행** 버튼을 클릭하거나 `Ctrl+Enter`를 누릅니다.
4. **도구 및 SEO**: 상단 도구 모음을 사용하여 OCR 도구, QR 코드 스캐너 또는 사이트맵/로봇 생성기에 액세스합니다.
5. **언어**: WebPad++는 30개의 글로벌 언어를 지원합니다! 즉시 전환하려면 오른쪽 상단의 드롭다운 메뉴를 사용하세요.

## 🔐 개인 정보 보호 및 보안

WebPad++는 전적으로 클라이언트 측에서 작동합니다. 귀하의 코드, 문서 및 데이터는 컴퓨터를 떠나지 않습니다. 모든 것은 브라우저의 기본 IndexedDB를 사용하여 지속적으로 저장됩니다. **모두 재설정**을 클릭하면 모든 데이터를 안전하게 지울 수 있습니다.
