## 🌟 Ikhtisar

WebPad++ adalah Integrated Development Environment (IDE) generasi berikutnya yang berjalan **100% di browser web Anda**. Tanpa memerlukan backend, database, atau instalasi lokal, Anda dapat menulis kode, menjalankan Python, membangun situs web, mengelola spreadsheet, dan mengekstrak teks dari PDF—semuanya secara lokal di mesin Anda tanpa latensi.

## ✨ Fitur Utama

### 💻 Pengeditan & Eksekusi Kode
- **Editor Cerdas**: Didukung oleh Ace Editor dengan penyorotan sintaksis, IntelliSense, dan pencocokan braket.
- **Eksekusi Instan**: 
  - Jalankan **HTML/CSS/JS** dengan Pratinjau langsung berdampingan.
  - Jalankan **Python** secara lokal langsung di browser melalui Pyodide.
- **Linter & Formatter Bawaan**: Deteksi kesalahan sintaksis real-time dan kecantikan sekali klik untuk HTML, CSS, dan JS.
- **Perbedaan Kode**: Bandingkan dua file secara berdampingan dengan penampil gabungan visual kami.

### 📄 Rangkaian Dokumen Profesional
- **Penurunan harga & Sinkronisasi Visual**: Beralih dengan mulus antara kode penurunan harga dan editor visual Rich Text (WYSIWYG).
- **Mesin Spreadsheet**: Membuka, mengedit, mengurutkan, dan menerapkan rumus matematika (misalnya, `=SUM(A1:A5)`) ke file `.xlsx` dan `.csv`.
- **Alat PDF & Word**: Lihat PDF, edit file DOCX dalam mode teks kaya, dan ekstrak teks melalui parser bawaan.
- **Mengekspor**: Konversi dan unduh file dalam format DOCX, XLSX, CSV, dan PDF.

### 🌐 Utilitas Tingkat Lanjut
- **Folder Drag & Drop**: Tarik seluruh direktori proyek dari OS Anda langsung ke WebPad++.
- **Alat SEO**: Generator visual bawaan untuk `sitemap.xml` dan `robots.txt` untuk meningkatkan peringkat penelusuran situs Anda.
- **OCR Gambar**: Unggah atau ambil foto dokumen dan ekstrak teks secara otomatis menggunakan Tesseract.js.
- **QR Code Suite**: Hasilkan kode QR secara instan atau pecahkan kodenya menggunakan webcam Anda.

## 🚀 Cara Menggunakan

1. **Memulai**: Cukup buka `index.html` di browser web modern mana pun.
2. **Manajemen File**: Klik menu `☰` untuk membuka File Explorer. Klik kanan untuk membuat file, atau cukup drag dan drop folder ke dalam jendela.
3. **Kode Berjalan**: Buka skrip apa pun dan klik tombol ▶️ **Jalankan** atau tekan `Ctrl+Enter`.
4. **Alat & SEO**: Gunakan bilah alat atas untuk mengakses alat OCR, pemindai Kode QR, atau generator Peta Situs/Robot.
5. **Bahasa**: WebPad++ mendukung 30 bahasa global! Gunakan menu tarik-turun di kanan atas untuk beralih secara instan.

## 🔐 Privasi & Keamanan

WebPad++ beroperasi sepenuhnya di sisi klien. Kode, dokumen, dan data Anda tidak pernah meninggalkan komputer Anda. Semuanya disimpan secara persisten menggunakan IndexedDB asli browser Anda. Anda dapat mengklik **Reset Semua** untuk menghapus semua data dengan aman.
