## 🌟 Обзор

WebPad++ — это интегрированная среда разработки (IDE) нового поколения, которая работает **100% в вашем веб-браузере**. Без необходимости использования серверной части, баз данных или локальных установок вы можете писать код, запускать Python, создавать веб-сайты, управлять электронными таблицами и извлекать текст из PDF-файлов — и все это локально на вашем компьютере с нулевой задержкой.

## ✨ Ключевые особенности

### 💻 Редактирование и выполнение кода
- **Интеллектуальный редактор**: на базе Ace Editor с подсветкой синтаксиса, IntelliSense и сопоставлением скобок.
- **Мгновенное выполнение**: 
  – Запустите **HTML/CSS/JS** с параллельным предварительным просмотром.
  - Запускайте **Python** локально прямо в браузере через Pyodide.
- **Встроенный Linter & Formatter**: обнаружение синтаксических ошибок в реальном времени и улучшение HTML, CSS и JS одним щелчком мыши.
- **Разница кода**: сравните любые два файла рядом с помощью нашей программы визуального слияния.

### 📄 Профессиональный пакет документов
- **Markdown и визуальная синхронизация**: плавное переключение между кодом Markdown и визуальным редактором форматированного текста (WYSIWYG).
- **Обработчик электронных таблиц**: открывайте, редактируйте, сортируйте и применяйте математические формулы (например, `=SUM(A1:A5)`) к файлам `.xlsx` и `.csv`.
- **Инструменты PDF и Word**: просматривайте PDF-файлы, редактируйте файлы DOCX в режиме форматированного текста и извлекайте текст с помощью встроенных анализаторов.
- **Экспорт**: конвертируйте и загружайте файлы в форматах DOCX, XLSX, CSV и PDF.

### 🌐 Расширенные утилиты
- **Перетаскивание папок**: перетаскивайте целые каталоги проектов из вашей ОС прямо в WebPad++.
- **Инструменты SEO**: встроенные визуальные генераторы файлов sitemap.xml и robots.txt для повышения рейтинга вашего сайта в поисковых системах.
- **Распознавание текста изображений**: загрузите или сделайте фотографию документа и автоматически извлеките текст с помощью Tesseract.js.
- **QR Code Suite**: мгновенно создавайте QR-коды или декодируйте их с помощью веб-камеры.

## 🚀 Как использовать

1. **Начало работы**: просто откройте index.html в любом современном веб-браузере.
2. **Управление файлами**: нажмите меню `☰`, чтобы открыть проводник. Щелкните правой кнопкой мыши, чтобы создать файлы, или просто перетащите папки в окно.
3. **Выполняемый код**: откройте любой скрипт и нажмите кнопку ▶️ **Выполнить** или нажмите «Ctrl+Enter».
4. **Инструменты и SEO**. Используйте верхнюю панель инструментов для доступа к инструменту OCR, сканеру QR-кода или генератору карты сайта/роботов.
5. **Языки**: WebPad++ поддерживает 30 глобальных языков! Используйте раскрывающееся меню в правом верхнем углу, чтобы мгновенно переключиться.

## 🔐 Конфиденциальность и безопасность

WebPad++ работает полностью на стороне клиента. Ваш код, документы и данные никогда не покидают ваш компьютер. Все хранится постоянно с использованием встроенной IndexedDB вашего браузера. Вы можете нажать **Сбросить все**, чтобы безопасно стереть все данные.
