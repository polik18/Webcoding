## 🌟 概述

WebPad++ 是下一代整合開發環境 (IDE)，**100% 在 Web 瀏覽器中運作**。無需後端、資料庫或本機安裝，您就可以編寫程式碼、運行 Python、建立網站、管理電子表格以及從 PDF 中提取文字 - 所有這些都在您的電腦上以零延遲本地完成。

## ✨ 主要特點

### 💻 程式碼編輯與執行
- **智慧型編輯器**：由 Ace Editor 提供支持，具有語法突出顯示、IntelliSense 和括號匹配功能。
- **即時執行**：
  - 透過即時並排預覽運行 **HTML/CSS/JS**。
  - 透過 Pyodide 直接在瀏覽器中本地運行 **Python**。
- **內建Linter & Formatter**：即時語法錯誤檢測和一鍵美化HTML、CSS和JS。
- **程式碼差異**：使用我們的視覺化合併檢視器並排比較任兩個檔案。

### 📄 專業文件套件
- **Markdown 和視覺同步**：在 Markdown 程式碼和富文本 (WYSIWYG) 視覺編輯器之間無縫切換。
- **電子表格引擎**：開啟、編輯、排序並將數學公式（例如「=SUM(A1:A5)」）套用到「.xlsx」和「.csv」檔案。
- **PDF 和 Word 工具**：檢視 PDF、以富文本模式編輯 DOCX 文件，以及透過內建解析器擷取文字。
- **匯出**：轉換並下載 DOCX、XLSX、CSV 和 PDF 格式的檔案。

### 🌐 進階實用程式
- **資料夾拖曳**：將整個專案目錄從作業系統直接拖曳到 WebPad++ 中。
- **SEO 工具**：內建「sitemap.xml」和「robots.txt」視覺產生器，可提高網站的搜尋排名。
- **圖像 OCR**：上傳或拍攝文件照片並使用 Tesseract.js 自動提取文字。
- **QR 代碼套件**：立即產生 QR 代碼或使用網路攝影機對其進行解碼。

## 🚀 如何使用

1. **入門**：只需在任何現代網頁瀏覽器中開啟「index.html」即可。
2. **檔案管理**：點選`☰`選單開啟檔案總管。右鍵單擊以建立文件，或只需將資料夾拖放到視窗中。
3. **執行程式碼**：開啟任意腳本，然後按一下 ▶️ **執行** 按鈕或按 `Ctrl+Enter`。
4. **工具和 SEO**：使用頂部工具列存取 OCR 工具、QR 碼掃描器或網站地圖/機器人產生器。
5. **語言**：WebPad++支援30種全球語言！使用右上角的下拉式選單進行即時切換。

## 🔐 隱私與安全

WebPad++ 完全在客戶端運作。您的程式碼、文件和資料永遠不會離開您的電腦。所有內容都使用瀏覽器的本機 IndexedDB 持久性儲存。您可以點擊“**全部重置**”以安全擦除所有資料。
