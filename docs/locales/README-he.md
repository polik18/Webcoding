## 🌟 סקירה כללית

WebPad++ הוא הדור הבא של סביבת פיתוח משולבת (IDE) הפועלת **100% בדפדפן האינטרנט שלך**. מבלי להזדקק ל-backend, מסדי נתונים או התקנות מקומיות, אתה יכול לכתוב קוד, להפעיל את Python, לבנות אתרים, לנהל גיליונות אלקטרוניים ולחלץ טקסט מקובצי PDF - הכל באופן מקומי במחשב שלך עם אפס השהייה.

## ✨ תכונות עיקריות

### 💻 עריכת קוד וביצוע
- **עורך חכם**: מופעל על ידי Ace Editor עם הדגשת תחביר, IntelliSense והתאמת סוגריים.
- **ביצוע מיידי**: 
  - הפעל את **HTML/CSS/JS** עם תצוגה מקדימה חיה זה לצד זה.
  - הפעל את **Python** באופן מקומי ישירות בדפדפן באמצעות Pyodide.
- **Linter & Formatter מובנים**: זיהוי שגיאות תחביר בזמן אמת וייפוף בלחיצה אחת עבור HTML, CSS ו-JS.
- **הבדל קוד**: השווה כל שני קבצים זה לצד זה עם מציג המיזוג החזותי שלנו.

### 📄 חבילת מסמכים מקצועית
- **Markdown & Visual Sync**: החלף בצורה חלקה בין קוד Markdown לבין עורך טקסט עשיר (WYSIWYG).
- **Spreadsheet Engine**: פתח, ערוך, מיין והחל נוסחאות מתמטיות (לדוגמה, `=SUM(A1:A5)`) על קבצי `.xlsx` ו-`.csv`.
- **PDF ו-Word Tools**: הצג קובצי PDF, ערוך קובצי DOCX במצב טקסט עשיר וחילוץ טקסט באמצעות מנתחים מובנים.
- **ייצוא**: המר והורד קבצים בפורמטים DOCX, XLSX, CSV ו-PDF.

### 🌐 כלי עזר מתקדמים
- **גרור ושחרר תיקיות**: גרור ספריות פרויקט שלמות ממערכת ההפעלה שלך ישר אל WebPad++.
- **כלי SEO**: מחוללים חזותיים מובנים עבור `sitemap.xml` ו-`robots.txt` כדי להגביר את דירוג החיפוש של האתר שלך.
- **OCR תמונה**: העלה או צלם תמונה של מסמך וחלץ אוטומטית את הטקסט באמצעות Tesseract.js.
- **QR Code Suite**: צור קודי QR באופן מיידי או פענח אותם באמצעות מצלמת האינטרנט שלך.

## 🚀 כיצד להשתמש

1. **תחילת העבודה**: פשוט פתח את 'index.html' בכל דפדפן אינטרנט מודרני.
2. **ניהול קבצים**: לחץ על תפריט `☰` כדי לפתוח את סייר הקבצים. לחץ לחיצה ימנית כדי ליצור קבצים, או פשוט גרור ושחרר תיקיות לתוך החלון.
3. **קוד ריצה**: פתח כל סקריפט ולחץ על כפתור ▶️ **הפעלה** או הקש `Ctrl+Enter`.
4. **כלים וקידום אתרים**: השתמש בסרגל הכלים העליון כדי לגשת לכלי OCR, סורק QR Code או מחולל Sitemap/Robots.
5. **שפות**: WebPad++ תומך ב-30 שפות גלובליות! השתמש בתפריט הנפתח בפינה השמאלית העליונה כדי לעבור באופן מיידי.

## 🔐 פרטיות ואבטחה

WebPad++ פועל כולו בצד הלקוח. הקוד, המסמכים והנתונים שלך לעולם לא יוצאים מהמחשב שלך. הכל מאוחסן באופן מתמיד באמצעות IndexedDB המקורי של הדפדפן שלך. אתה יכול ללחוץ על **אפס הכל** כדי למחוק את כל הנתונים בצורה מאובטחת.
