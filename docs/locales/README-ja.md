## 🌟 概要

WebPad++ は、**100% Web ブラウザで実行**できる次世代の統合開発環境 (IDE) です。バックエンド、データベース、またはローカル インストールを必要とせず、コードの作成、Python の実行、Web サイトの構築、スプレッドシートの管理、PDF からのテキストの抽出を、すべて遅延ゼロでローカルのマシン上で行うことができます。

## ✨ 主な特徴

### 💻 コードの編集と実行
- **インテリジェント エディタ**: 構文の強調表示、IntelliSense、ブラケット マッチングを備えた Ace Editor を搭載。
- **即時実行**: 
  - **HTML/CSS/JS** をライブの並列プレビューで実行します。
  - Pyodide を介して **Python** をブラウザ内でローカルに直接実行します。
- **組み込みのリンターとフォーマッタ**: リアルタイムの構文エラー検出と HTML、CSS、および JS のワンクリック整形。
- **コードの差分**: ビジュアルマージビューアを使用して、任意の 2 つのファイルを並べて比較します。

### 📄 プロフェッショナル ドキュメント スイート
- **マークダウンとビジュアル同期**: マークダウン コードとリッチ テキスト (WYSIWYG) ビジュアル エディターをシームレスに切り替えます。
- **スプレッドシート エンジン**: `.xlsx` および `.csv` ファイルを開いて、編集、並べ替え、数式 (例: =SUM(A1:A5)`) を適用します。
- **PDF および Word ツール**: PDF を表示し、リッチ テキスト モードで DOCX ファイルを編集し、内蔵パーサーを介してテキストを抽出します。
- **エクスポート**: DOCX、XLSX、CSV、PDF 形式でファイルを変換してダウンロードします。

### 🌐 高度なユーティリティ
- **フォルダー ドラッグ アンド ドロップ**: プロジェクト ディレクトリ全体を OS から WebPad++ に直接ドラッグします。
- **SEO ツール**: サイトの検索ランキングを高めるための `sitemap.xml` および `robots.txt` 用のビルトイン ビジュアル ジェネレーター。
- **画像 OCR**: ドキュメントの写真をアップロードまたはスナップし、Tesseract.js を使用してテキストを自動的に抽出します。
- **QR コード スイート**: QR コードを即座に生成するか、Web カメラを使用してデコードします。

## 🚀 使用方法

1. **はじめに**: 最新の Web ブラウザで「index.html」を開くだけです。
2. **ファイル管理**: `☰` メニューをクリックしてファイル エクスプローラーを開きます。右クリックしてファイルを作成するか、フォルダーをウィンドウにドラッグ アンド ドロップします。
3. **コードの実行**: 任意のスクリプトを開いて、[▶️ **実行**] ボタンをクリックするか、`Ctrl+Enter` を押します。
4. **ツールと SEO**: 上部のツールバーを使用して、OCR ツール、QR コード スキャナー、またはサイトマップ/ロボット ジェネレーターにアクセスします。
5. **言語**: WebPad++ は 30 の世界言語をサポートしています。右上のドロップダウン メニューを使用して、すぐに切り替えます。

## 🔐 プライバシーとセキュリティ

WebPad++ は完全にクライアント側で動作します。コード、ドキュメント、データがコンピューターから離れることはありません。すべてはブラウザのネイティブ IndexedDB を使用して永続的に保存されます。 [**すべてリセット**] をクリックすると、すべてのデータを安全に消去できます。
