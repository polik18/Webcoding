## 🌟 সংক্ষিপ্ত বিবরণ

WebPad++ হল একটি পরবর্তী প্রজন্মের ইন্টিগ্রেটেড ডেভেলপমেন্ট এনভায়রনমেন্ট (IDE) যা আপনার ওয়েব ব্রাউজারে **100% চালায়**। ব্যাকএন্ড, ডেটাবেস বা স্থানীয় ইনস্টলেশনের প্রয়োজন ছাড়াই, আপনি কোড লিখতে পারেন, পাইথন চালাতে পারেন, ওয়েবসাইট তৈরি করতে পারেন, স্প্রেডশীটগুলি পরিচালনা করতে পারেন এবং PDF থেকে পাঠ্য বের করতে পারেন—সবকিছুই আপনার মেশিনে স্থানীয়ভাবে শূন্য লেটেন্সি সহ।

## ✨ মূল বৈশিষ্ট্য

### 💻 কোড এডিটিং এবং এক্সিকিউশন
- **বুদ্ধিমান সম্পাদক**: সিনট্যাক্স হাইলাইটিং, ইন্টেলিসেন্স এবং ব্র্যাকেট ম্যাচিং সহ Ace এডিটর দ্বারা চালিত৷
- **তাত্ক্ষণিক সম্পাদন**: 
  - লাইভ পাশাপাশি প্রিভিউ সহ **HTML/CSS/JS** চালান।
  - Pyodide এর মাধ্যমে সরাসরি ব্রাউজারে স্থানীয়ভাবে **Python** চালান।
- **বিল্ট-ইন লিন্টার এবং ফরম্যাটার**: রিয়েল-টাইম সিনট্যাক্স ত্রুটি সনাক্তকরণ এবং এইচটিএমএল, সিএসএস এবং জেএস-এর জন্য এক-ক্লিক বিউটিফাই।
- **কোড ডিফ**: আমাদের ভিজ্যুয়াল মার্জ ভিউয়ারের সাথে যেকোনো দুটি ফাইল পাশাপাশি তুলনা করুন।

### 📄 প্রফেশনাল ডকুমেন্ট স্যুট
- **মার্কডাউন এবং ভিজ্যুয়াল সিঙ্ক**: মার্কডাউন কোড এবং একটি রিচ টেক্সট (WYSIWYG) ভিজ্যুয়াল এডিটরের মধ্যে নির্বিঘ্নে টগল করুন।
- **স্প্রেডশীট ইঞ্জিন**: গাণিতিক সূত্রগুলি খুলুন, সম্পাদনা করুন, সাজান এবং প্রয়োগ করুন (যেমন, `=SUM(A1:A5)`) `.xlsx` এবং `.csv` ফাইলগুলিতে৷
- **PDF এবং Word টুলস**: PDF দেখুন, DOCX ফাইল রিচ-টেক্সট মোডে এডিট করুন এবং বিল্ট-ইন পার্সারের মাধ্যমে টেক্সট বের করুন।
- **রপ্তানি**: DOCX, XLSX, CSV, এবং PDF ফর্ম্যাট জুড়ে ফাইলগুলিকে রূপান্তর এবং ডাউনলোড করুন৷

### 🌐 উন্নত ইউটিলিটি
- **ফোল্ডার টেনে আনুন এবং ড্রপ করুন**: আপনার OS থেকে সরাসরি ওয়েবপ্যাড++-এ সম্পূর্ণ প্রোজেক্ট ডিরেক্টরি টেনে আনুন।
- **SEO টুলস**: আপনার সাইটের সার্চ র‍্যাঙ্কিং বাড়াতে `sitemap.xml` এবং `robots.txt`-এর জন্য অন্তর্নির্মিত ভিজ্যুয়াল জেনারেটর।
- **ইমেজ OCR**: একটি নথির ফটো আপলোড করুন বা স্ন্যাপ করুন এবং Tesseract.js ব্যবহার করে স্বয়ংক্রিয়ভাবে পাঠ্যটি বের করুন।
- **QR কোড স্যুট**: অবিলম্বে QR কোডগুলি তৈরি করুন বা আপনার ওয়েবক্যাম ব্যবহার করে সেগুলিকে ডিকোড করুন৷

## 🚀 কিভাবে ব্যবহার করবেন

1. **শুরু করা**: যেকোনো আধুনিক ওয়েব ব্রাউজারে সহজভাবে `index.html` খুলুন।
2. **ফাইল ম্যানেজমেন্ট**: ফাইল এক্সপ্লোরার খুলতে `☰` মেনুতে ক্লিক করুন। ফাইল তৈরি করতে ডান-ক্লিক করুন, অথবা উইন্ডোতে ফোল্ডারগুলিকে টেনে আনুন এবং ফেলে দিন।
3. **রানিং কোড**: যেকোনো স্ক্রিপ্ট খুলুন এবং ▶️ **রান** বোতামে ক্লিক করুন বা `Ctrl+Enter` টিপুন।
4. **টুল এবং এসইও**: OCR টুল, QR কোড স্ক্যানার বা সাইটম্যাপ/রোবট জেনারেটর অ্যাক্সেস করতে শীর্ষ টুলবার ব্যবহার করুন।
5. **ভাষা**: ওয়েবপ্যাড++ ৩০টি বৈশ্বিক ভাষা সমর্থন করে! অবিলম্বে স্যুইচ করতে উপরের ডানদিকে ড্রপডাউন মেনু ব্যবহার করুন।

## 🔐 গোপনীয়তা এবং নিরাপত্তা

WebPad++ সম্পূর্ণভাবে ক্লায়েন্ট-সাইড পরিচালনা করে। আপনার কোড, ডকুমেন্ট এবং ডেটা কখনই আপনার কম্পিউটার ছেড়ে যায় না। আপনার ব্রাউজারের নেটিভ ইনডেক্সডডিবি ব্যবহার করে সবকিছু অবিরামভাবে সংরক্ষণ করা হয়। নিরাপদে সমস্ত ডেটা মুছে ফেলার জন্য আপনি **রিসেট সব** এ ক্লিক করতে পারেন।
